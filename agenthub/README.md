# AgentHub - Projektin Aivot

Tämä kansio sisältää tekoälyagenttien älykkyyden, roolit ja säännöt.

## Hakemistorakenne:
* `personas/`: Agenttien roolit, tavoitteet ja sielunelämä (kuka agentti on).
* `rules/`: Agenttien yhteiset ja ehdottomat säännöt, mm. virheistä oppiminen.
* `workflows/`: Askel-askeleelta ohjeet tiettyjen tehtävien suorittamiseen (Miten asiat tehdään).
